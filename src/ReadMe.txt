================================================================
■ ハンゲームアバターキャッシュクリーナー 1.0 by ななこっち★ ■
================================================================

ハンゲームのかんたんゲームで蓄積されるアバターキャッシュの
肥大化を防止する為に、パソコン起動時に自動削除を行うものとして
作成致しました。

ハンゲームの動作不安定になるのを回避する他、（そこまで極端にありませんが）
ハンゲームをしていくうちに、勝手にディスクが満タンになるのを防ぎます。

※じっくりゲームやブラウザーゲームでは、意味がありません。

■ インストール
スタートアップにインストールされます。

■ 使用方法
パソコンを起動時に、自動的に起動します。

■ アンインストール
スタートメニュー→コントロールパネル→プログラムと・・・
のあたりのメニューから、ハンゲームアバターキャッシュクリーナーを
選択すれば、アンインストールができます。

レジストリ等は何もいじっていません。

■ 注意
初回起動は、アバターキャッシュが大量に保存されている
可能性があるため、非常に時間がかかります。
そのため、初回起動の時にHDD等のアクセスランプが点灯したまま
特に何も表示していないのは仕様です。

■ 動作内容
単純に、レジストリを見て、ハンゲームかんたんゲームの
Avatar フォルダを削除しているだけですが、高速削除されるように、
一度リネームを行ない、フォルダを作成してから、リネームを行なった
フォルダを削除しています。

このアプリケーションは、スタートアップ時は、何も
エラーメッセージ等を出力しません。
ハンゲームをアンインストールしても、自動実行されますので、
必要に応じてアンインストールが必要になります。

■ 動作チェック環境
Windows XP 32bit / Windows 8 64bit上で行なっています。

■ 非常に参考にしたもの
どこか・・・

ソースをそのままパクッている状態と同じ状態です。

■ ライセンス
一切著作権を主張しません。フリーソフトウェアではなく、
PDS（パプリックドメインソフトウェア）として扱います。

ただし、アイコンについては、NHN JAPAN の著作物になります。
